# CCFDML
**Credit Card Fraud Detection Dataset**
**Overview**
This repository contains ml and dl models for detecting credit card fraud transactions which uses a dataset for Credit Card Fraud Detection, sourced from Kaggle. The dataset is provided by the Machine Learning Group at ULB (Université Libre de Bruxelles). It is essential for developing and testing machine learning models to detect fraudulent credit card transactions.

Dataset Link: Credit Card Fraud Detection Dataset (https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data)

**Dataset Description**
The dataset consists of transactions made by credit cards in September 2013 by European cardholders. It contains both legitimate and fraudulent transactions, making it suitable for building and evaluating fraud detection models. The features in the dataset have been anonymized due to privacy concerns, and the dataset is highly imbalanced, with a small percentage of fraudulent transactions.

**Files**
creditcard.csv: The main dataset file in CSV format, containing transaction details.
